CPU/Assembler upgrades.

November 20th, 2022

Hello! o/

I've had this bad habit where I will stay up realllyyy late at night working on homework. In fact, on Friday I stayed up until 1:30am working on my CPU project. I restarted the project from scratch and made a new circuit, with a more organized and readable structure. Unfortunately, I can't share much in terms of screenshots as the project isn't due yet. After it's done, I might upload it to my GitHub page (including the Logisim circuit file, assembler, and language instructions). Be sure to check that out often to see all of my personal projects!

Since, I can't share pictures of my project, I will instead post a picture of something a received in the mail last week: my Irish passport! My dad and his family were born and raised in Ireland, and they moved to the United States when he was 13. Ireland's citizenship laws are very cool; children of Irish-born citizens are also automatically Irish citizens, and therefore eligible to apply for a passport. So, that's what I did! Not that I'm doing any traveling any time soon, but it's always nice to have the benefits that come with being a dual citizen.

Okay, now back to CS stuff! Originally I designed the langauge around a 16-bit machine code, but quickly realized that if I wanted to add more functionality to the CPU, I would need to increase the machine code to 32 bits. Now, the language supports loading and storing from memory, as well as adding, subtracting, multiplying, and dividing 8-bit numbers. More instructions will be added soon, as well as an upgrade to a pipelined datapath.

I also did some more research on Python functions that I can use to make a more capable assembler. I wish I could, but I can't share details of my assembler implementation until after I submit the project (in case anyone I know from school ends up reading this). The project is due December 5th (finished). See me flexing my JavaScript skills? Be impressed!

I have an exam early tomorrow morning, so I better get to bed. Good night!

Building a CPU and an assembler.

November 17th, 2022

For today's picture: what the dog doin.

This is my dog, Maya! I miss her a lot when I'm at school. She misses me a lot too, and always gets really excited when I come home for the weekend. I'm going home on a Sunday this week so I'll see her soon!

I've gotten a head start on a project for my Computer Architecture class: building a CPU, creating an assembly language, and an assembler to process instructions. Before you ask, no, its not a real CPU; it's a virtual one. Our task was to build a "CPU" in Logisim (a program for building and simulating circuits and logic gates), create our own assembly language and machine code, and also an assembler program to translate from assembly to machine code.

Surprisingly, I had a lot of fun with this project. Most of my friends and classmates would dread having to do schoolwork in their free time, and I thought I would feel the same. However this semester I've been enjoying the content of my classes a lot more. Right now I am taking Computer Architecture and Algorithms. Aside from a couple of topics, I've been able to easily absorb and apply the content of these classes. All of the programming assignments are really fun for me, and I think I've really developed a passion for coding! Even though I'm just typing words like an essay right now, I'm having fun because there is a bunch of code on my screen! Recently, I've been doing a few small side-projects in Python. One program was a test script for an extra credit assignment, which would run C++ code and test to see if the output was correct. Another program, which I am currently working on, is an assembler script, to translate my assembly language into a machine code image file to be used in Logisim (see image for example). I'm going to start doing more projects in Python soon, especially over winter break once I have more time.

That's all for today's post, I'll be sure to post more updates soon about the CPU project and my other personal projects!

My first blog!

Welcome to my blog! I've never really written a blog before or posted on the internet ever, so this is all very new to me. (I think the most blogging I've done is serially-posting on Twitter when I first got an account back in middle school. It was a very cringe account!) But anyways, this will be my first "post" on my first blog.

The main purpose of this page is to, first and foremost, practice HTML and CSS programming. The best way to learn is by doing, so I am doing! I feel rather behind on my skills compared to some of my classmates, and I want to expand those skills and work my way up towards more complex projects. This blog will be my first true personal programming project that I've worked on. (As such, please don't expect any fancing formatting or features in these early days; I'm still learning. But, do check back often, as I will update formatting and features as I learn more!

What to include in my first blog...? I guess an introduction would be nice! My name is John (o/ hi!). I'm currently a sophomore (almost junior) Computer Science student at Stevens Institute of Technology (in Hoboken, NJ). I was born and raised in a small suburb in northern New Jersey. Computers have always been a hobby for me; while my friends would ask for new shoes and such for birthdays, I would ask for new tech and gadgets to entertain myself with. Unfortunately, my school's computer science offerings were extremely lacking (perhaps that's a topic for another blog post!), but ever since starting at Stevens in 2021, I've been slowly and surely expanding my skillset.

I might be late to the game when it comes to learning HTML and CSS, but better late than never! I'm not sure yet how frequently I will post or what kinds of things I will write about. Keep reading to find out!
